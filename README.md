# Study Tracker

学習進捗を記録・可視化できる Web アプリです。

[こちら](https://study-tracker.shunniehub.com/)から見ることができます。

---

## アプリ紹介

![alt text](/public/image.png)

主な機能

- 学習記録の登録
- 教材登録・編集・削除
- 学習記録一覧の表示・削除
